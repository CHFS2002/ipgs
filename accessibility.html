Accessibility Statement

IPGS is committed to making its website usable by the widest possible audience.

Conformance target

This website targets Web Content Accessibility Guidelines (WCAG) 2.1, Level AA.

What we have done

  • Semantic HTML5 landmarks, ARIA where needed, and proper heading hierarchy.
  • Visible focus states on all interactive elements; full keyboard navigation.
  • Text color contrast ratios meet or exceed AA thresholds.
  • Text can be resized to 200 % without layout breakage or loss of information.
  • All content images use descriptive alt text; decorative images are marked empty.
  • Language is declared per page (lang="en" / lang="zh-CN").
  • Reduced-motion preferences are respected.

Known exceptions

As of the initial launch, we are aware of the following limitations:

  • PDF attachments (e.g., the Charter) may lack full accessibility tagging pending professional remediation.
  • Some third-party embeds used on events pages (maps, calendar) have independent accessibility profiles.

Report an issue

If you encounter any accessibility barrier on this site — a keyboard trap, contrast failure, missing alt text, or a feature that does not work with your assistive technology — please let us know:

  • Email: chfsfao@mail.hzau.edu.cn
  • Via the general-enquiry form, selecting "Other" and noting "Accessibility".

We aim to acknowledge within five working days and to fix confirmed issues within a reasonable timeframe.

Review

This statement is reviewed annually, and after any substantial change to the website. A designated Associate Member accessibility panel audits the site each quarter.
